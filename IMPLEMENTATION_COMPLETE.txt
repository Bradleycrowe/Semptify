KEY FEATURES
================================================================================

MODALS & TRIGGERS:
  - View Lease Agreement
  - View Eviction Notice
  - Send Formal Notice
  - Request Notarization
  - Track Certified Mail
  - Switch Language

VOICE COMMANDS:
  - Speak any trigger phrase after clicking "Start Listening"
  - App automatically opens matching modal with help text
  - Example: "View Lease Agreement" opens lease modal

MULTILINGUAL SUPPORT:
  - English (en)
  - Español (es)
  - Somali (som)
  - Hmong (hmn)

  Each modal displays help text in selected language.

ACCESSIBILITY:
  - Text-to-speech for all modal content
  - Voice recognition for voice commands
  - Language selector for accessibility
  - Keyboard navigation compatible
