Persistent Storage Required

‚ùå Application Startup Blocked
Semptify requires persistent storage to function. The application will not start without it.
‚ö†Ô∏è Why Storage Matters
Without persistent storage, ALL user data is lost on container restart:
  • Vault documents - PDFs, images, lease agreements, evidence
  • User accounts - Login credentials, authentication tokens
  • Timeline history - Important dates and events
  • Notary certificates - Document integrity proofs
  • Admin access - Administrative credentials
  • AI learning - Adaptive system knowledge

Configuration Options

1. Cloudflare R2

Cost: ~$2-3/month for typical usage
Setup: 10 minutes

Set these environment variables in your deployment platform:

  • R2_ACCOUNT_ID - Your Cloudflare account ID
  • R2_ACCESS_KEY_ID - API access key
  • R2_SECRET_ACCESS_KEY - API secret
  • R2_BUCKET_NAME - Bucket name (e.g., "semptify-vault")
  • R2_ENDPOINT_URL - Your R2 endpoint URL

2. Google Cloud Storage

Cost: Free tier available (5GB)
Setup: 15 minutes

Requirements:

  • Install: google-cloud-storage package
  • GOOGLE_APPLICATION_CREDENTIALS - Service account key path
  • GCS_BUCKET_NAME - Bucket name
  • Upload service account JSON key as secret file

3. Render Persistent Disk

Cost: $10/month for 10GB
Setup: 5 minutes

Configuration:

  • Create persistent disk in Render dashboard
  • Set PERSISTENT_DISK_PATH=/mnt/data
  • Restart application
