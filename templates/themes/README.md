# Semptify UI Themes

Three responsive GUI themes created for different use cases:

## Theme 1: Legal Document Portal
**File:** `templates/themes/theme1_legal_portal.html`
**Style:** Formal, serif fonts, document-focused
**Colors:** Navy (#1a2332) + Muted Gold (#c9a961)
**Best for:** Professional legal document management, court filings, formal correspondence
**Features:** 
- Fixed header with subtle navigation
- Hero section with centered messaging
- Grid layout for document categories
- High contrast for readability
- Fully responsive (mobile-first)

## Theme 2: Tenant Helpdesk
**File:** `templates/themes/theme2_helpdesk.html`
**Style:** Friendly, modern, support-focused
**Colors:** Indigo (#4f46e5) + Cyan (#06b6d4) + Green (#10b981)
**Best for:** Help center, FAQs, guided workflows, community support
**Features:**
- Emergency hotline banner
- Gradient welcome hero
- Quick action cards with CTAs
- Icon-based help tiles
- Sticky header navigation
- Fully responsive

## Theme 3: Rights Action Dashboard
**File:** `templates/themes/theme3_action_dashboard.html`
**Style:** Urgent, dark mode, action-oriented
**Colors:** Red (#dc2626) + Amber (#f59e0b) + Emerald (#059669) on dark (#18181b)
**Best for:** Crisis management, deadline tracking, urgent actions, case monitoring
**Features:**
- Urgent alert bar
- Status cards with metrics
- Priority-tagged action items
- Activity timeline
- High-contrast dark theme
- Uppercase typography for emphasis
- Fully responsive

## Usage
Open any theme file directly in a browser to preview. Each is a complete standalone HTML page.
