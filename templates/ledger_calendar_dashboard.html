⚖️ Ledger & Calendar Dashboard

API Endpoints

The Ledger & Calendar system provides REST API endpoints for tracking events and legal timeline management.

Available Endpoints:

  • GET /api/ledger-calendar/ledger - View ledger entries
  • GET /api/ledger-calendar/calendar - View calendar events
  • POST /api/ledger-calendar/ledger - Add new ledger entry
  • POST /api/ledger-calendar/calendar - Schedule new event

Query Parameters:

  • type - Filter by entry type (document, payment, complaint, evidence, notice, action)
  • actor - Filter by actor/user
  • start_time - Start timestamp
  • end_time - End timestamp
  • limit - Max results (default: 100)
