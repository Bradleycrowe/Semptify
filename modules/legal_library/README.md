# Legal Library Module

Structured, versioned repository of tenant-facing legal references. Provides authoritative, plain-language summaries mapped to official government sources.

## Goals
- Authority: Only verifiable .gov / official judiciary sources.
- Clarity: Plainâ€‘language summaries + glossary.
- Maintainability: Scripted validation & update workflows.
- Auditability: CHANGELOG versioning with semantic increments.

## Structure
- references/: Source JSON files + aggregated references.json
- glossaries/: Tenant-friendly term definitions
- workflows/: PowerShell maintenance scripts
- assets/: Badges, visual indicators, future diagrams

## JSON Schema (per entry)
`json
{
  "id": "unique-kebab",
  "category": "federal|state|tax|funding|other",
  "title": "Human readable title",
  "summary": "Plain-language tenant-focused summary",
  "source_url": "https://...",
  "jurisdiction": "US|MN|IRS|Federal|...",
  "updated": "YYYY-MM-DD",
  "tags": ["keyword1", "keyword2"],
  "risk_level": "low|medium|high",
  "recommended_action": "Suggested tenant action",
  "disclaimer": "Non-legal advice disclaimer",
  "version": 1
}
`
